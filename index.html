# 这是一个复杂多变的世界

梦境

我曾经在一个梦境中看见过一只变幻无常的白色小猫，一开始瞳孔是蓝色的，下一秒又变成红色，它的尾巴异常的大，宛如一只狐狸。除了瞳孔意外，整个身子都在不停的变化，只有这条尾巴，一直保持着原来的样子。

这个世界就跟这只小猫一样，从你出生的那一刻起，她就在不停的变化着，尚且不谈论你出生前跟离开人世后的时间，对你有意义的时间只有你的有生之年。

我们都活在不断变幻的复杂世界里，即使你一直不停地探索世界的真相，但其实你也只不过是在沙漠里一颗颗沙子地数着，想把沙漠的沙子数完，那是不可能的事情。所以在你的有生之年，应该以什么样的姿态活着，可能会是一个比较重要的话题。

是的，世界在变幻，我愿意成为那只小白猫的尾巴，蓬松地竖立在够不着尽头却还要不断探索的世界上，告诉自己，我愿意这样活着。
