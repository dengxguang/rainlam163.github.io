# 一些高三的人

2010-08-10

我已经高中毕业两个月，在家霉了一个月，在广州又霉了一个月。我那些懒散的日子已经随着高三的结束消失不见了，那些每天无论在校园哪个角落都看得见我的游荡的影子的日子再也没有了，写日记的习惯也没了，上课玩手机的日子也消逝了，晚睡溜出去上网通宵的机会也没有了，与通通穿拖鞋走天下的梦想也看不见了。我的头脑已经发热，甚至许多事情许多人都已经记不清了。我在想，是不是再过一个月，我就会忘记所有关于高三的人，我每天都在回忆，但是总有一天，尽管不是我想，我都会忘记你们。趁着现在头脑还有一点发热，我想记录我现在所能记住的人。

# 〔壹〕老师

# 1、欧阳

当然第一个想到的是欧阳，因为这是个很有特色的班主任。此人教物理。你拿一些浅的题目去问他，他会说，这些题目扫街的阿姨都会，怎么你就不会呢？你拿一些相对有难度的题目去，他就会说，这些题目……你不用钻研太深，不用理它了。事实上他自己也没有能力解答这个题目。另外，欧阳的普通话说得真的很普通，某日上公开课，欧阳用普通话上，结果就让人笑掉大牙。欧阳是最和蔼可亲的班主任，基本上他是不管班上的纪律的，我们隔天就会扣分，只有当他心情不好时才会show show你。欧阳是全校出了名的翘课王，甚至比我们学生翘的课还多。每每连堂的课，第一节他会来上，第二节他就在级室看他的NBA。他是一个NBA的超级粉丝。宿舍也经常不查，我们每天晚上都有人溜出去是因为知道欧阳不会来，即使来了也不清点人数。我们四班有超过二十个人复读，创造了纪录，我想说，高三这一年不要怪欧阳对我们做了什么，其实他什么也没做。嘿嘿，欧阳真是一个有趣的家伙。

# 2、潇洒哥

潇洒哥真的很潇洒。此人教数学，走起路来面带笑容，春风满面，没剪头发之前的发型很有型，印象中他貌似没发过火。巡班的时候只巡他的三班，路过我们四班也从没把目光移进过我们班。此人在级室喜欢用电脑看女明星，当然，这是男人的嗜好，但是，潇洒哥能做到在众学生面前毫不收敛，这就体现了他的潇洒之处，这是所有男人都不得不佩服的。

# 3、语文婆

此人就给我的最大印象是她的高跟鞋。她的高跟鞋没什么特别，但是穿在她的脚上就会变得很有趣，当她从级室过来的时候总是未见其人先闻其拖鞋声，所以在她眼里，我们都很守纪律，因为在她来之前我们一听到她“笃笃”的高跟鞋声，我们就会装作很认真学习的样子。嘿嘿。她总是让我们不停地做做阅读，每节课都做，印象的高三语文课除了做阅读还是做阅读，所以我们都很怕上语文课。另外，此人在级室喜欢偷菜，我一直以为，这种无聊的行为只有我们学生在学习无聊之余才会做，现在我看到的是，上课也很无聊。

# 4、英语婆

我不知道为什么总要在老师的称呼上总要加个“婆”或“佬”字，但是大家都这么叫，我也跟着叫了。此人给我的印象是，全世界都欠她的钱以致于她要时常以一副恶霸的表情对着全世界。事实上她天生就这表情。我想她老公一定很有钱，我经常见她穿新衣服，几乎没什么重复。所谓千金难买美人笑，虽然她不是美人，但是她的笑也是千金难买。你偶然看见她对你笑了，首先得想想你是否做了什么坏事，如果没有，那更糟糕。此人的口头禅是“莫名其妙”。此人在级室的时候喜欢看欧美电影看美剧，没办法，谁叫她是教英语的。

# 5、坤哥

这是唯一一个我敬佩的高三老师。他已年近六十，教化学，在全校都有很好的口碑。除了有点老懵懂上课经常说错东西，普通话说得不怎么样（但是比欧阳好多了，起码响应了国家政策，用普通话讲课，欧阳讲课都用家乡话）之外，还是一个很好的老师及领导。此人备课很认真，讲课很认真，都是拼了命的，所以让人很佩服。美中不足的是，由于年老，什么都慢人一等，所以总是拖课，一个星期下来，就比别人上多了一节课。

# 6、生物婆

鄙人被她教了三年生物，结果什么也没学到，但这是我的问题，没天赋，对生物一窍不通。她的名字是让人终身难忘的，车全球，踩着单车跑遍全球。但是，她的性格很和蔼可亲，每个人都可以亲她，通通说得对，就一个典型的善良勤劳的中国妇女。上课照着书本读，生怕你看不懂上面的字，所以她上的课其实十分轻松。

# 〔贰〕朋友

# 1、通通

通通是个废仔，有关于他的一切已经在我的日志《兄弟成长于天蓝时代01—波罗通》中详细描述。这是个痴情男子，对燕子有契而不舍的精神，虽然燕子早已舍弃他。他是个懂得欣赏音乐的人，我们一起逛商场，一起用奶瓶喝水，一起穿拖鞋走天下，一起下午下课后散步到学校门外吃雪糕，一起翘晚睡到网吧通宵，一起在高四的楼梯口看MM……我不想再说一遍，值得回忆的我忘不了，这是我高中所接触过的唯一一个受了外界影响还能保持童真的男孩子。欲要了解通通，请看我的日记《兄弟成长于天蓝时代01—波罗通》。

# 2、八婆

八婆一称呼来源于通通之口，年龄上是90后的大姐大，外貌上是00后的小姐小。此人经常迷恋东北大汉，身高的问题还是不要在此曝光，一来身高是女孩的秘密，二来我曝光了就没命了。后来她说喜欢学校里我一哥们，于是我就成了媒人，她整天想着我那哥们都想疯了，还说要嫁给他和他生小孩，我晕～

# 3、汪叔

此人和我同班三年，和省委书记汪洋同姓同名，经常被人叫书记。和我稔好，每见到我总说，芳，我爱你！PS：我被他叫小芳，不知道为什么，估计小芳是她的初恋女友，把他飞了，就把我想象成她。此人说话幽默风趣，说笑自己不笑，是幽默高手。

# 4、KG

KG长得很粗犷，满脸胡须，很man很男人。对电脑颇有研究，对电影也颇有研究，曾经不到半价卖了OPPO S7i给我，嘿嘿。此人很会储钱，通常我们的钱都不够用，但是他的钱，十几块几块地储起来，很快就有钱买高达了，这让我很佩服。

# 5、苏俊豪

原谅我只能写你的真实名字，貌似你没化名。但是我们同班四年吧？此人是魔兽高手，CS高手，延至CF都是高手。但是，他的学习是很好的，这让我十分不爽，凭什么我们都翘晚睡上网通宵，他的学习可以好，我们就不行呢？

………………

关于类似的人还有很多，我已经写得要抓狂了，但是，你知道，有些东西已经深深印在了我的心里，就不会忘记的。感谢你们。

对于即将去读大学的朋友，其实哪间学校都一样，你要过的充实就好，不能和高三那样碌碌无为了。对于读高四的朋友，你们已经落后了一年，你们得努力，不要读高五，这条路你们走两年，应该比我们这些走一年的有更好的结果。
