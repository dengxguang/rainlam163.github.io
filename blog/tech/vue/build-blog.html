# 使用 VuePress 搭建个人博客

# VuePress 是什么

VuePress 实际上可以看作是一个静态网站生成器，设计之初是为了支持 Vue 及其子项目的文档需求，因为支持 Markdown ，所以我们用它来搭建自己的个人博客也是非常适合的，尤大大也表示 VuePress 将来会发展成为一个博客系统。具体的有关 VuePress 的介绍，可以到 VuePress官网 查看。

# 为什么选择 VuePress

基本上你只需要关注你书写的博客内容就可以，对于喜欢使用 Vue 的开发人员来说更是一个好消息，因为 VuePress 支持你使用 Vue 组件，你可以把你对于 Vue 的习惯无缝地迁移到 VuePress 上。
